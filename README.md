# DSA Visualizer

An interactive Data Structures & Algorithms visualizer built with HTML, CSS, and JavaScript.

## Features

- **Sorting Visualizer** — Bubble, Selection, Insertion, Merge, Quick Sort with animated bars, pseudocode panel, and a live editable code editor that highlights the currently executing line
- **Pathfinding Visualizer** — BFS, DFS, Dijkstra's on an interactive grid with wall drawing and maze generation
- **BST Visualizer** — Binary Search Tree with insert, delete, search, and inorder/preorder/postorder traversal animations

## How to Run

1. Download or clone this repository
2. Open `index.html` in any browser
3. No installation or server required
